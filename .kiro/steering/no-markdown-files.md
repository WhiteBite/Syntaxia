# Правило: Не создавать Markdown файлы

**ЗАПРЕЩЕНО** создавать .md файлы для документирования работы без явного запроса пользователя.

## Запрещено

- SUMMARY.md, REPORT.md, CHANGES.md, NOTES.md
- Любые .md файлы для отчётов о работе
- Документирование выполненных действий в markdown

## Разрешено

- Создавать .md только по явному запросу пользователя
- Редактировать существующие .md (README.md, документация)

## Вместо этого

Кратко сообщить о выполненной работе в ответе чата.
